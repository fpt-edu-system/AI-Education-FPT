# Hướng dẫn Đóng góp (Contributing Guidelines)

Cảm ơn bạn đã đóng góp cốt lõi cho hệ sinh thái AI FPT! 

## 1. Mở Issue trước khi Code
Tất cả các thay đổi phải được link tới 1 thẻ Issue hợp lệ (FEAT, SUPL, BUG) trên GitHub.
Khuyến khích bạn clone luôn nhánh `feature/xxx` mà hệ thống đã chuẩn bị sẵn.

## 2. Quy chuẩn tạo PR (Pull Requests)
- Không đẩy thẳng vào nhánh `main` (Nhánh này đã được khóa bảo vệ).
- Code phải vượt qua bài kiểm duyệt (Lint/Test) trên GitHub Actions.
- Cần ít nhất 1 thành viên Approve (Phê duyệt) thì code của bạn mới được sát nhập.

## 3. Quy ước Đặt tên Commit
Follow chuẩn Conventional Commits:
- `feat: thêm chức năng chấm AI`
- `fix: sửa lỗi crash UI`
- `docs: cập nhật README`
